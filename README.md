# Flux Bot

Telegram бот для гарантирования сделок между пользователями.

## Возможности

- ✅ Создание сделок (обычные товары и Telegram Gifts)
- ✅ Оплата звездами или валютой (RUB/USD/EUR)
- ✅ Реферальная система (1% от сделок рефералов)
- ✅ Админ-панель (бан пользователей, просмотр сделок, разрешение споров)
- ✅ Система споров с доказательствами от обеих сторон

## Требования

- Python 3.12+
- Telegram бот токен

## Использование

### Главное меню

- **Добавить реквизиты карты** - добавить реквизиты для ручного перевода
- **Создать сделку** - создать новую сделку
- **Мои сделки** - просмотр своих сделок
- **Реферальная ссылка** - получить реферальную ссылку
- **Поддержка** - связаться с поддержкой

### Создание сделки

1. Выберите тип сделки (обычный товар или Telegram Gift)
2. Введите название товара
3. Выберите способ оплаты (звезды или валюта)
4. Укажите цену
5. Отправьте ссылку покупателю

### Процесс сделки

**Товар:**
1. Покупатель оплачивает боту
2. Продавец передает товар
3. Покупатель подтверждает получение
4. Деньги переводятся продавцу

**Telegram Gift:**
1. Покупатель оплачивает боту
2. Продавец нажимает "Я передал подарок"
3. Продавец отправляет подарок покупателю напрямую в Telegram
4. Покупатель подтверждает "Это тот подарок" или открывает спор

### Система споров

При отказе от подарка или товара:
1. Покупатель отправляет описание проблемы с доказательствами (фото/документы)
2. Продавец может ответить своими доказательствами
3. Админ рассматривает спор и решает - вернуть деньги или выплатить продавцу

### Админ-панель

Доступ по команде `/admin`:

- Просмотр всех сделок
- Просмотр сделок конкретного пользователя
- Бан/разбан пользователей
- Разрешение споров

При завершении сделки админ видит реквизиты продавца.
При отмене сделки админ видит реквизиты покупателя.

### Статусы сделок

- **pending** - ожидает присоединения покупателя
- **joined** - покупатель присоединился
- **paid** - оплата получена
- **gift_sent** - продавец подтвердил передачу подарка
- **gift_received** - подарок получен покупателем
- **completed** - сделка завершена
- **cancelled** - сделка отменена
- **disputed** - спор требует решения админа

## Автоудаление

Сделки со статусом pending/joined старше 24 часов удаляются автоматически.
